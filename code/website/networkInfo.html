Mainnet:

The main Ethereum network.

Ropsten test network:

Launched in 2016 to replace the Morden test network. It was attacked in February 2016 and taken down, and with great effort was revived in March 2017.

Kovan test network:

A proof-of-authority (PoA) testnet issued by Ethcore, Melonport, and Digix after the Ropsten attacks.

Rinkeby test network:

Another proof-of-authority (PoA) testnet.

Unknown/test network:

This is usually a testing network run through localhost.
